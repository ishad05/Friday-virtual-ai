# Friday – Virtual AI Assistant

Friday is a Python-based virtual AI assistant inspired by Tony Stark’s FRIDAY, designed to provide intelligent, contextual, and personality-driven conversations using AI models. This specific model is trained to give savage replies but it can be trained to have any personality, creator desires using livekit agents

The project combines a python backend with a simple frontend interface, demonstrating the practical application of conversational AI in a real system.

## Features
- AI-powered conversational assistant
- Python-based backend logic
- API-driven AI interaction
- Simple frontend interface
- Modular and extensible architecture

## Tech Stack
- Python
- OpenAI API
- JavaScript (Frontend)
- Typescript
- Framework - React, Tailwind
- REST-style API communication
- Livekit Agents

## Getting Started

### Prerequisites
- Python 3.8 or higher
- Node.js
- OpenAI API key

### Installation
```
git clone https://github.com/ishad05/Friday-virtual-ai.git
cd Friday-virtual-ai
```

Install Python dependencies:
```
pip install -r requirements.txt
```

Install frontend dependencies:
```
npm install
```

### Running the Project
```
python main.py
npm start
```
